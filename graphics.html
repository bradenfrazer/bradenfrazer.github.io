Graphics

I've always taken an interest in graphics, whether it's mashing things up in Photoshop or creating vector-based designs in Illustrator. With the help of an Intro to Graphic Design class, I've been able to hone my skills in both of those programs. Below you'll find the majority of my work from the semester I took that class, as well as some other graphics-related projects I've worked on.

Risky Ribbit

Risky Ribbit is an Android game developed in Unity with a team for my Senior capstone project at Texas A&M. For the project, I designed all of the UI, graphical assets, and menus used in the game.

Night Owls

Night Owls is a fictional card game that I designed in a Graphic Design class that I took. As part of the project, I designed the vectorized logo, business cards and letter head for the company, and packaging.

Archer brochure

Typographic brochure for the font Archer created as a graphic design project.

Typcycle Font Face

Font face designed by me using basic geometry and a grid.

Ben Rector concert poster

Poster designed for a Ben Rector concert using my font face, Typcycle.

Convolution poster

Miscellaneous poster designed using my font face, Typcycle.

Cartoon Monster

A halloween-themed cartoon monster created for fun in a graphic design class.

Apotheosis Concert Poster

Poster advertisement created for a concert for Apotheosis, a Texas A&M Men’s Acapella Group.
